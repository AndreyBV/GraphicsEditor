# GraphicsEditor

A graphics editor that allows you to: draw and move lines, use morphing for lines, apply transformation operations using matrices. There is also a functional for drawing a fractal tree.

## Initialization

1. [Download this repository](https://github.com/AndreyBV/GraphicsEditor/archive/refs/heads/main.zip);
1. Open the project file **"GED.sln"**;
1. Run the project **"Ctrl+F5"**.

## Base function

- Drawing lines **(Keyshort: Ctrl+2)**;
- Single movement of drawn lines **(Keyshort: Ctrl+1)**;
- Selection of multiple lines **(Keyshort: V)**;
- Moving multiple lines
- Application of transformation matrix for selected lines;
- Moving any one end of the line;
- Removing lines **(Keyshort: Ctrl+Z or Right mouse click)**;
- Performing the morphing operation to the selected lines (lines are selected in pairs, between which the morphing will be performed);
- Morphing stage control (0 - no morphing, 1 - maximum morphing effect);
- Adding new canvases and performing basic operations on them;
- Saving lines drawn on canvases to a file with the possibility of subsequent loading;
- Construction of a fractal tree with a choice of construction parameters.
